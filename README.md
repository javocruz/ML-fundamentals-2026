# ML-fundamentals-2026

ML Fundamentals Project 1

## How to run

1. Install dependencies: 
    pip install -r requirements.txt

2. Open the notebook: 
    jupyter notebook assignment1_Javier_Cruz.ipynb

3. Run all cells (or run them one by one).

The notebook uses `bank-additional.csv` in the same folder. Make sure you're in the repo root when you run it.
